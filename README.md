# SHBFinance-Loan-Data-Analysis-and-Customer-Segmentation
Phân tích dữ liệu khoản vay và khách hàng SHBFinance giai đoạn 2022–2023, bao gồm xử lý dữ liệu, phân tích chân dung khách hàng, đặc điểm khoản vay, xu hướng tăng trưởng và phân khúc khách hàng bằng K-means để đề xuất chiến lược tăng khoản vay và bán chéo.
